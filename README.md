# Музыкальное лото

Веб-приложение для ведущего музыкального лото: 90 окошек, по клику воспроизводится песня.

## Возможности

- Сетка **9×10 = 90 окошек** с номерами 1–90
- Каждое окошко показывает **статичное превью** YouTube-видео
- **Клик по окошку** — запускает видео со звуком (только одно одновременно)
- Отметка сыгранных песен (галочка после окончания трека)
- Кнопка «Новая игра» — сброс состояния
- Редактор песен: название + URL для каждого номера
- Импорт и экспорт списка песен в JSON
- Сохранение настроек в `localStorage` браузера

## Запуск

### Требования

- [Node.js](https://nodejs.org/) 18+ (для локального сервера разработки)

### Установка и запуск

```bash
cd C:\Users\skyca\musical-lotto
npm install
npm run dev
```

Откройте в браузере адрес, который покажет Vite (обычно `http://localhost:5173`).

### Сборка для публикации

```bash
npm run build
npm run preview
```

Готовые файлы появятся в папке `dist/`.

## Как добавить песни

1. Нажмите **«Настройки песен»**
2. Для каждого номера укажите **название** и **ссылку на YouTube**
3. Нажмите **«Сохранить»** — видео появится в окошках и начнёт крутиться без звука

### Какие ссылки работают

**YouTube** — обычная ссылка на видео:

```
https://www.youtube.com/watch?v=dQw4w9WgXcQ
https://youtu.be/dQw4w9WgXcQ
https://music.youtube.com/watch?v=dQw4w9WgXcQ
```

**MP3** (опционально) — при клике воспроизводится аудио без фонового видео.

**Не работают** ссылки на VK, Яндекс.Музыку и другие сервисы.

## Формат JSON

Файл должен содержать ровно 90 объектов:

```json
[
  { "id": 1, "title": "Владимирский централ", "url": "https://www.youtube.com/watch?v=XXXXXXXXXXX" },
  { "id": 2, "title": "Мурка", "url": "https://example.com/2.mp3" }
]
```

Используйте **«Экспорт JSON»** для резервной копии и **«Импорт JSON»** для загрузки готового списка.

## Как играть

1. Ведущий видит 90 окошек с крутящимся без звука видео
2. Нажимает на номер — **включается звук** у этого окошка
3. Участники отмечают песню на своих бланках
4. Окошко помечается как сыгранное
5. «Выключить звук» — глушит все окошки
6. «Новая игра» — сброс отметок для нового раунда
